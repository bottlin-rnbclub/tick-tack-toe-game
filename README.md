# tick-tack-toe-game

## Установка проекта:

## Перед установкой проверьте наличие:
- Python 3.6 or higher
- `tkinter` library
- `numpy` library
- Если они не установлены, то можно установить их через pip

### Эти зависимости можно установить с помощью `pip`:

```
git clone my-project
pip install tk, numpy
python main.py
```
